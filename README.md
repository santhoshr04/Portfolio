## Local Development

### Requirements

- Node.js
- npm
- Git

### Installation

```bash
npm install
```

### Environment Variables

Create a `.env` file in the project root:

```env
VITE_DISCORD_WEBHOOK_URL=your_discord_webhook_url
```

The webhook is used by the portfolio contact form. Keep the real value private and never commit it to source control.

### Start the Development Server

```bash
npm run dev
```

The application is available at [http://localhost:5173](http://localhost:5173).

### Create a Production Build

```bash
npm run build
```

### Preview the Production Build

```bash
npm run preview
```
